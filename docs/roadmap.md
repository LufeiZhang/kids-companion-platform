# 产品路线图

## V1 MVP（当前）

- 三角色登录与权限
- 教师创建房间、邀请已分配学生
- 教师/学生课堂主界面
- Socket.IO 信令与 ACK
- 按页互动白板、课件上传/翻页
- 四类奖励动画、专注提醒
- 页面可见性与离线状态
- PostgreSQL 日志与管理后台
- 可替换 RTC 抽象（音视频仍为占位）

下一步上线前优先：真实 RTC、自动化测试、Redis 在线状态、对象存储、家长授权记录、安全加固和可观测性。

## V2 学习管理增强

- 日/周/月任务计划和番茄钟
- 积分、徽章、任务完成度
- 教师查看任务进度
- 课堂事件回放与课后报告
- 经评估和授权的 AI 学习助手
- 多学生课堂、分组活动、运营统计

## V3 Android 深度能力

- Android 原生壳与稳定 RTC
- 家长授权下的屏幕共享与悬浮涂鸦层
- 专用平板模式、设备管理员、防误退出
- 护眼距离检测与强制休息

强管控功能必须单独完成儿童权益、设备权限、家长同意与可撤回机制评审，不应沿用 MVP 的普通 Web 授权假设。
